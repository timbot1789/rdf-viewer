# RDF Viewer

A simple RDF viewer written in Vue.
Core Libraries:
- (Nuxt 3)[https://nuxt.com/]
- (two.js)[https://two.js.org/]
- (rdflib)[https://github.com/linkeddata/rdflib.js/]

## Setup

You will need Node Version 16 or higher. NOTE: you will need to install Node version on the system. Nuxt currently does not interface correctly with NVM.

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```

## Production

Build the application for production:

```bash
# npm
npm run build

# pnpm
pnpm run build

# yarn
yarn build
```

Locally preview production build:

```bash
# npm
npm run preview

# pnpm
pnpm run preview

# yarn
yarn preview
```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
